## Pre-requisites

- [Node.js](https://nodejs.org/en/) (LTS / Tested with v20.9.0)
- [Yarn](https://yarnpkg.com/) (LTS / Tested with v1.22.21)

## Quickstart

1. Install dependencies:

    ```bash
    yarn install
    ```

1. Run local chain / network:

    ```bash
    yarn chain
    ```

1. Deploy contracts:

    ```bash
    yarn deploy
    ```

1. Start Debug-Site:
    
    ```bash
    yarn debug:start
    ```
    Visit: http://localhost:3001

1. Start Frontend:

    ```bash
    yarn start
    ```
    Visit: http://localhost:3000
